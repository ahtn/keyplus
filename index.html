Ease of use

keyplus is easy to configure. The layout is stored in an easy-to-read YAML file, and there is a simple GUI to program the keyboard.

Wireless support

keyplus was built with wireless in mind. It supports wireless split and regular wireless. Just about every possible configuration you can think of is compatible. keyplus uses a Logitech Unifying receiver, so your mouse and keyboard can be connected to the same dongle.

keyplus mini

The keyplus mini is the flagship keyplus-compatible device, featuring row/column labeling, USB Type C, and a connection for a battery.

Get keyplus today.

The keyplus GitHub page has all the information you need to get started. If you have a keyplus mini, you can jump right in. Otherwise, check out the supported hardware list to see if you have a compatible device.
